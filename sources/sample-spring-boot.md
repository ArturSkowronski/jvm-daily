# Spring Boot 4.0 RC1 Released

Spring Boot 4.0 Release Candidate 1 is now available. Key highlights:

- Virtual threads enabled by default
- GraalVM native image support improvements
- New observability features with Micrometer 2.0
- Kotlin 2.x full support
- Jakarta EE 11 baseline

The team recommends testing migration from 3.x early. Breaking changes include removal of deprecated properties and updated minimum Java version to 21.
